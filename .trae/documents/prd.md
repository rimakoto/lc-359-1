## 1. 产品概述

单位换算工具箱是一款面向日常烹饪、学习、工作等场景的实用工具应用。解决菜谱上毫升/克、旅行时英里/公里等跨单位换算困扰，让用户一键完成精准换算。
- 核心价值：消除单位换算的认知负担，提供即时、直觉化的换算体验
- 目标用户：烹饪爱好者、留学生、旅行者、工程师等需要频繁进行单位换算的人群

## 2. 核心功能

### 2.2 功能模块

1. **主页面**：类别标签栏、双输入框换算区、单位翻转按钮、收藏列表

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主页面 | 类别标签栏 | 展示长度、重量、体积、温度、面积五个类别标签，点击切换类别 |
| 主页面 | 双输入框换算区 | 上下两个输入框，上方输入原始数值+选择原始单位，下方自动显示目标数值+选择目标单位，任意一端输入另一端实时联动 |
| 主页面 | 单位翻转按钮 | 中间翻转按钮，点击交换上下输入框的单位，数值随之重新计算 |
| 主页面 | 收藏功能区 | 收藏按钮将当前换算组合保存；收藏列表展示已保存组合，点击直接加载 |
| 主页面 | 单位选择器 | 下拉选择器展示当前类别下所有可用单位 |

## 3. 核心流程

用户打开应用 → 选择换算类别 → 在任一输入框输入数值并选择单位 → 另一输入框实时显示换算结果 → 可翻转单位方向 → 可收藏常用换算组合 → 下次点击收藏快速加载

    "收藏当前组合?" --> "否: 继续使用"

    "交换上下单位, 重新计算" --> "继续输入"

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深靛蓝 (#1e1b4b) 搭配琥珀色 (#f59e0b) 强调色，温暖而精准
- 按钮风格：圆角按钮，翻转按钮带旋转动画
- 字体：使用 DM Serif Display 作为展示字体，DM Sans 作为正文字体
- 布局：居中卡片式布局，上方类别标签，中间双输入框区域，下方收藏列表
- 图标：使用 lucide-react 图标库

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 主页面 | 类别标签栏 | 水平滚动的胶囊标签，选中态带琥珀色底色，字体 DM Sans Semibold |
| 主页面 | 双输入框换算区 | 玻璃质感卡片容器，上下两个大号输入框+单位下拉，中间翻转按钮带箭头图标 |
| 主页面 | 翻转按钮 | 圆形按钮，hover 时旋转 180°，点击时弹性缩放动画 |
| 主页面 | 收藏区域 | 收藏按钮（心形图标），收藏列表以小卡片网格展示，hover 微浮起 |

### 4.3 响应式设计

- 桌面优先设计，最大宽度 640px 居中
- 移动端适配：类别标签水平可滚动，输入框全宽
- 触摸优化：按钮和选择器最小点击区域 44px

### 4.4 动效设计

- 类别切换：卡片内容淡入淡出过渡
- 翻转动画：上下输入框交换位置 + 旋转图标
- 收藏动画：心形按钮点击弹跳 + 收藏卡片滑入
- 数值变化：数字滚动过渡效果
